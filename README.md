## Get Started

### Prerequisites

* OS: Linux
* Golang: v1.12+

### Installation

#### Clone

* Clone this repo to your local machine using https://github.com/amazingchow/engine-vector-space-search-service.git.

#### Setup

```shell
# build the binary
make build

# start the service
./engine-vector-space-search-service
```
